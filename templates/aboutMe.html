I’m a passionate developer who loves building end-to-end IoT solutions and data dashboards. With experience in Python, Flask, AWS, and embedded MicroPython, I turn sensor data into actionable insights.

When I’m not coding, you’ll find me hiking, experimenting with Raspberry Pi projects, or sketching UI designs.

My educational path begun in a rather traditional way, initially i pursued my Media Technology degree by enrolling the programme Interactive Media and Web Technologies at Linnaeus University.

However, at the end of my 2nd year I felt out of place. I began catching interest in cyber security, cloud computing, server configuration and backend development. Hence I decided to drop out of the programme and instead tailor my own Bachelor's Degree in Media Technology.

My Projects

Description

A real-time IoT dashboard that collects temperature and humidity readings from a DHT11 sensor connected to my laptop, stores them in MySQL, and visualizes the last five measurements (updated every 5 seconds) alongside historical trends.

Tech Stack

Backend: Python, Flask, MySQL Frontend: Chart.js, Bootstrap 5, plain JavaScript (Fetch API) Hosting: AWS EC2 (Amazon Linux), Route 53, Elastic IP Tooling: python-dotenv for configuration, GitHub Actions for CI/CD

Course / Origin

Part of the course Introduction to Applied Internet of Things at Linnaeus University.
